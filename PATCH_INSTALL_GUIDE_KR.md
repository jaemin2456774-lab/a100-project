# Railway Patch Install Guide

1. Back up the current repository state.
2. Overwrite only the files included in this patch.
3. Do not remove `/data`, Runtime Evidence, Ledger, Learning, or configuration files.
4. Deploy on Railway and perform one clean restart.
5. Run: `/version`, `/versionaudit`, `/runtimehealth`, `/status`, `/commandmatrix`, `/crossengineaudit`, `/evidencereplay`, `/rcpreflight`, `/releasegate`, `/errors`.
