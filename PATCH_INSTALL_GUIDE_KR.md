# S52 증분 패치 설치 안내

1. Railway 연결 GitHub 저장소의 기존 `main.py`를 패치 파일로 덮어씁니다.
2. 커밋 후 Railway 배포 완료 및 startup preflight PASS를 확인합니다.
3. 기존 `/data`, 환경변수, 설정, 학습 데이터는 삭제하거나 초기화하지 않습니다.
