1. ZIP을 GitHub 저장소 루트에 덮어씁니다.
2. Railway에서 새 배포를 실행합니다.
3. 아래 명령을 순서대로 확인합니다.

/version
/buildinfo
/routeraudit
/status
/runtimehealth
/verifyall
/verifyall detail
/connectivity detail
/errors
