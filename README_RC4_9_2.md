# A100 V116.0 LTS RC4.9.2
Execution-Based Command Functional Certification Development Release

설치 후 확인 명령:

```
/version
/versionaudit
/commandcert
/ltscert
/repositoryaudit
/healthscore
/pipelineaudit
```

`/commandcert`의 PARTIAL은 기능 고장을 의미하지 않으며, 안전한 실행 Adapter가 아직 등록되지 않아 실제 Engine 실행 증거가 부족한 명령을 의미합니다. FAILED는 Handler 단절을 의미합니다.
