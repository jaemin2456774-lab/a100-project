# A100 V93.1 Self-Learning AI

V93.0을 기준으로 기존 schema 1의 Paper/Shadow 종료 결과를 읽어 자동 재학습하는 독립 분석 계층입니다.

## 핵심 개선
- 종료 결과 기반 예상승률 보정 및 95% 신뢰구간
- LONG/SHORT, 시장 국면, 종목, Paper/Shadow별 메모리 통계
- 엔진 점수 스냅샷이 있는 표본에서 엔진별 성과를 학습해 가중치 자동 보정
- AI 설명에 지지/억제 엔진, 유사 표본 승률, 보정 예상승률 표시
- 기존 `/aicore`, `/aistatus`, `/aiperformance`, `/aiweights`, `/aimemory` 명령을 V93.1로 교체
- `/help`, `/commands V93` 단일 레지스트리 자동 동기화

## 데이터 안전
- 상태 파일: `a100_v91_paper_state.json` 유지
- schema: 1 유지
- 별도 상태 스키마 및 실주문 경로 추가 없음
- 학습은 기존 종료 데이터에서 실행 시 계산하므로 데이터 마이그레이션 불필요

## 참고
과거 버전 테스트 중 `V91_VERSION == V92.x`처럼 당시 최종 버전을 정확히 고정한 검사는 새 버전에서 의도적으로 실패합니다. V93.1 전용 테스트와 최종 preflight를 기준으로 검증합니다.
