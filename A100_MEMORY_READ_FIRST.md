# A100 PROJECT MEMORY — READ FIRST

This directory is an official part of the A100 source repository.

Before changing code or proposing a roadmap change:
1. Read `PROJECT_CONSTITUTION.md`.
2. Read `MASTER_ROADMAP.md`.
3. Read `ACTIVE_SPRINT.md`.
4. Read `DECISION_LOG.md`.
5. Add new ideas to `FUTURE_QUEUE.md`; do not redirect the active sprint.
6. Update `CHAT_HANDOFF.md` before moving to a new chat.

These files are project memory, not optional release notes.
