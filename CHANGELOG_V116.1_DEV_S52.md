# A100 V116.1 DEV S52 변경 사항

## 핵심 수정
- `/sniper` LONG/SHORT/WAIT 시각 막대 그래프 복구
- `/sniper detail` 방향 기여도, Consensus Tree, Decision Quality, Evidence Coverage 분리 출력
- `/ultimate` 모바일 요약형 대시보드 적용
- `/ultimate detail`에서만 전체 근거 출력
- 이모지 기반 상태 구분과 중복 문구 축소
- Evidence Coverage의 UNKNOWN/Age 반복을 N/A 중심으로 단순화

## 보존 사항
Runtime First, Strict Read Only, Evidence Only, Registry 341, Schema 1, Paper 20, Shadow 60, Live Trading OFF, Synthetic PASS OFF, Gate 계산식 불변, 기존 데이터/설정 보존.
